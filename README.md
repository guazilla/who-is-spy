# 谁是卧底 AI 自动裁判系统 (Who is the Spy - AI Judge System)

这是一个基于大语言模型（LLM）的自动化《谁是卧底》对战平台。该程序扮演“自动化法官”（Game Engine）的角色，提供一个深色现代风格的桌面图形界面（GUI），自动驱动 4 个（可相同或不同）的大模型作为玩家进行对局。

系统全程自动分发身份、管理游戏状态、构建上下文，并调用 OpenRouter API。在对局结束后，还会自动让 AI 玩家根据全局上帝视角生成赛后复盘总结。

## ✨ 核心特性

- **🤖 全自动多 Agent 对局**：支持混搭不同的模型，自动分配平民词和卧底词，AI 玩家全自动进行发言、推理、伪装和投票。
- **🎨 现代深色 GUI 界面**：基于 PyQt6 打造，提供顺滑的流式打字机动画输出效果，游戏进程清晰明了。
- **⚡ 异步与并发处理**：采用多线程与并发队列处理 API 请求，确保界面流畅不卡顿；并行生成复盘总结，大幅缩短赛后等待时间。
- **🎮 完整的游戏机制支持**：支持 AI 玩家进行战术选择：
  - `NEXT`：正常发言结束交接回合。
  - `VOTE`：发起投票，打断其他玩家的后续发言权。
  - `BOOM`：卧底确认平民词后发起自爆，猜词抢夺胜利。
- **🧠 精彩的赛后复盘**：游戏分出胜负后，AI 会拿到包含真实身份的全景实况记录，站在上帝视角总结自己的战术得失，并评价其他玩家的表现。
- **🛡️ 容错与配置管理**：支持自定义 OpenRouter API Base URL；网络错误或超时时提供弹窗重试机制。

## 🛠️ 技术栈

- **语言**: Python 3
- **界面**: PyQt6
- **网络与 API**: openai (官方 SDK 兼容 OpenRouter), httpx
- **并发控制**: threading, concurrent.futures

## 📦 安装与运行

1. **克隆项目**
   ```bash
   git clone https://github.com/guazilla/who-is-spy.git
   cd whoisspy
   ```

2. **安装依赖**
   建议使用虚拟环境：
   ```bash
   pip install -r requirements.txt
   ```
   
3. **初始配置**
   - 重命名config.json.example为config.json，并在配置文件中输入你的 **OpenRouter API Key**。
   - 点击保存后即可开始对局。

3. **运行程序**
   ```bash
   python main.py
   ```

## 🕹️ 使用指南

1. **设定词语**：在界面右侧输入本次对局的“平民词”和“卧底词”。
2. **选择模型**：点击左侧玩家面板，为 1-4 号玩家分别指定参赛的模型。
3. **开始游戏**：点击“开始游戏”，引擎会在后台随机抽取卧底身份，游戏正式开始。
4. **日志保存**：对局结束及复盘完成后，完整的日志会自动保存在 `logs/` 目录下（例如：`game_公交车_地铁_时间戳.txt`）。
